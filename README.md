# App.ir is a Dashboard app

## Setup

    1. Run - cd tool.db
    2. create appConfig.json file with below format
                {
            "port" : "",
            "baseUrl" : "",
            "dbConfig" : {
                "dbName" : "",
                "userName" : "",
                "connectString" : "",
                "password" : ""
            }
        }

    3. Run - npm i
    4. Run - npm start | npm run dev (dev mode)
